# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-20

### Added
- Initial extraction from ImuChat web-app monolith
- Games Hub with featured carousel, category filter, and game cards
- Anime Quiz game with timer, scoreboard, and animations
- i18n support (EN, FR, JA)
- ImuChat SDK integration via `ImuChatProvider`
- Standalone development mode (fallback without SDK)
- Tailwind CSS with ImuChat CSS custom properties theming
- Lightweight UI components (Button, Card, Badge, Avatar, Progress)
